Supplementary Materials for:
"Economic Narrative Indices: A Systematic Review of Sentiment-Based Forecasting"
—— Submitted to Journal of Economic Surveys ——

This folder contains the following supplementary files:

Manuscript and title files:
   - Manuscript_JES.pdf: anonymized manuscript for double-blind review.
   - Manuscript_JES.tex: editable LaTeX source for the manuscript.
   - Title_Page_JES.txt: separate title page with author details and required
     policy statements.
   - Cover_Letter_JES.txt: cover letter.

1. literature_matrix.xlsx
   - Full literature matrix for all 66 papers included in the systematic review.
   - 4 sheets: Coded Matrix, Audit Summary, Coding Guide, Source Notes.
   - The Coded Matrix includes one row per reviewed paper and audit columns for
     primary method, OOS testing, real-time protocol, DM tests, MCS/SPA tests,
     benchmark type, standardized RMSE reduction, region, and language.

2. papers_database.csv
   - Machine-readable flat-file version of the literature matrix.
   - 66 rows, 19 columns including relevance score (1-5).
   - Suitable for import into statistical software for quantitative synthesis.

3. results_data_raw.csv
   - Numerical data extracted from paper results for quantitative synthesis.
   - Includes effect sizes, forecast horizons, methodological descriptors.
   - Used for the within-group standardized effect-size analysis in the manuscript.

4. quality_assessment.xlsx
   - Risk-of-bias and methodological quality assessment for each paper.
   - Contains exactly 66 scored paper rows plus a separate audit-summary sheet.
   - Three scoring dimensions (Methodology 0-3, Data Quality 0-2, Validation 0-2)
     with Combined Score 0-7.
   - Mean quality score: 4.2/7 (Moderate).

5. Figures/
   - figure1_timeline.png: Timeline of methodological evolution (2007--2025).
   - figure2_performance_distribution.png: Distribution of RMSE reductions across domains.
   - figure3_geographic_distribution.png: Geographic concentration of reviewed studies.
   - figure4_prisma.png: PRISMA flow diagram for study selection.

6. Tables/
   - table1_metasummary.tex / .png: Summary of key metrics across all 66 papers.
   - table2_examples.tex / .png: Representative papers across methodological spectrum.
   - table3_validation_by_era.tex / .png: Validation practices by publication period.
   - table4_quality_appendix.tex / .png: Quality assessment summary (Appendix).
