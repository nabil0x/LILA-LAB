## SUBMISSION TIMELINE & STRATEGY

### Option A: SEQUENTIAL (SAFEST)
```
Week 1-2:   Submit to JES (primary)
            ↓ (Wait 6-9 months)
Month 6-9:  Receive decision from JES
            ├─ IF ACCEPT: Done! Celebrate 🎉
            ├─ IF MINOR REVISIONS: Revise & resubmit
            └─ IF REJECT: 
                ↓
Month 10:   Submit to IJF (faster review, 3-4 months)
                ↓
Month 13:   Receive IJF decision
            ├─ IF ACCEPT: Done! 🎉
            └─ IF REJECT: Submit to JEL (next cycle)
```

**Pros**: Each submission tailored perfectly; time to improve based on feedback  
**Cons**: Very slow; might take 12-18 months total

### Option B: PARALLEL (RISKY but FASTER)
```
Week 1:     Prepare 2 versions
            ├─ JES version (9,000 words, all tables, theory-focused)
            └─ IJF version (9,000 words, validation-focused)

Week 2-3:   Submit JES + IJF simultaneously
            ↓ (4-9 months wait)

Month 4-9:  Receive decisions
            ├─ IF EITHER ACCEPTS: Withdraw from others, submit there
            ├─ IF BOTH REJECT: Submit to JEL
            └─ IF ONE ASKS FOR REVISIONS: Revise, resubmit

```

**Pros**: Faster total time (can publish in 6-12 months if lucky)  
**Cons**: Risk of simultaneous acceptance (have to withdraw); journal editors may object

### Option C: RECOMMENDED (BALANCED)
```
Week 1:     Prepare master version (current 9,000 words)

Week 2:     Submit to JES (best fit, highest probability)
            Keep IJF version ready

Month 2-3:  While waiting for JES initial screening:
            ├─ Minor tweaks for IJF version
            └─ Prepare JEL stripped-down version (8,200 words)

Month 4-6:  IF JES not accepted yet, submit to IJF
            (Parallel submission OK if JES still in review)

Month 9:    JES makes decision
            ├─ IF ACCEPT: Done!
            └─ IF REJECT: IJF decision likely also here
                ├─ IF IJF ACCEPTS: Done!
                └─ IF IJF REJECTS: Submit to JEL
```

**Best for**: Maximizing acceptance probability with reasonable speed

## COVER LETTER TEMPLATES

### JES Cover Letter
```
Dear Editor,

We submit for review: "Economic Narrative Indices and Media-Based 
Sentiment Measures: A Systematic Review of Methodologies, Applications, 
and Research Gaps (2007–2025)"

This paper synthesizes evidence from 66 peer-reviewed studies on 
sentiment-based economic forecasting. Our key contributions are:

1. First meta-analysis establishing 20% median RMSE improvement
2. Methodological taxonomy comparing dictionary, ML, BERT, and LLM approaches
3. Rigorous quality assessment of 66 papers using risk-of-bias framework
4. Documentation of critical geographic (56% US) and linguistic (84% English) gaps
5. Novel best-practices validation framework (Tier 1/2/3 system)
6. Policy applications for central banks and emerging markets

The paper is 9,000 words, includes 6 tables and 3 figures, and addresses 
an important gap in economic sentiment literature. It is suitable for 
Journal of Economic Surveys and is submitted exclusively here.

Sincerely,
[Authors]
```

### IJF Cover Letter
```
Dear Editor,

We submit: "Sentiment-Based Economic Forecasting: A Systematic Review 
of Validation Practices, Performance Metrics, and Best Practices for 
Forecasters (2007–2025)"

For practitioners and forecasters, this review provides:

1. Quantified performance: 20% median RMSE improvement over benchmarks
2. Validation audit: Only 33% of papers follow real-time data protocols
3. Methods comparison: Clear trade-offs between dictionary/ML/BERT/LLM
4. Best-practices framework: Tiered validation standards for implementation
5. Practical recommendations: Decision tree for practitioners choosing methods

We analyze 66 high-quality studies, assess 3 validation approaches, and 
provide actionable guidance for real-time forecasting deployment. The paper 
is 9,000 words with emphasis on practical methodology validation.

Sincerely,
[Authors]
```

### JEL Cover Letter
```
Dear Editor,

We submit: "The Sentiment-Based Forecasting Literature: Unresolved 
Questions, Geographic Gaps, and a Critical Assessment of Evidence"

This survey critically examines 66 studies on sentiment-based economic 
forecasting, identifying key unresolved questions:

1. CAUSALITY: Do sentiments cause outcomes or merely reveal them earlier?
   → Current literature provides only Granger causality; no causal ID
   
2. INFORMATION vs. NOISE: Are sentiment-driven price movements persistent
   or reverting? → Evidence mixed; context-dependent
   
3. GEOGRAPHIC GAPS: 56% of studies focus on USA; zero coverage of 
   Africa/Latin America/Bangla-speaking regions
   
4. VALIDATION INCONSISTENCY: Only 33% follow real-time protocols; 
   publication bias likely inflates reported improvements

We develop a novel best-practices validation framework and propose 
research roadmap for closing identified gaps.

Sincerely,
[Authors]
```

## SUBMISSION PLATFORM GUIDES

### JES (ScholarOne Manuscripts)
1. Go to: https://mc.manuscriptcentral.com/jes
2. Register: Create account
3. Submit: "New Manuscript"
4. Upload: PDF manuscript
5. Enter: Abstract, keywords, JEL codes
6. Suggest: 5-6 reviewers (optional)
7. Assign: Editor (auto-assigned)
8. Track: "View Decision" page

**Timeline**: Desk review (1-2 weeks) → Sent to reviewers (varies) → Decision (6-9 months)

### IJF (Editorial Manager)
1. Go to: https://www.editorialmanager.com/ijf/
2. Register: Create account
3. Submit: "New Manuscript"
4. Upload: PDF + any supplementary files
5. Enter: Title, abstract, keywords
6. Select: Research area (Time series forecasting)
7. Suggest: Reviewers
8. Track: "Status" page

**Timeline**: Initial review (1 week) → Reviewers (2-3 weeks) → Decision (3-4 months)

### JEL (Editorial Express)
1. Go to: https://www.editorialexpress.com/
2. Register: Create account under "Journal of Economic Literature"
3. Submit: "New Submission"
4. Upload: PDF manuscript
5. Enter: Abstract, keywords
6. Suggest: 4-5 reviewers (IMPORTANT)
7. Add: Research classification
8. Track: Submission status

**Timeline**: Screening (1-2 weeks) → Reviewers (4-8 weeks) → Decision (4-6 months)

## FINAL CHECKLIST BEFORE SUBMISSION

**FOR ANY JOURNAL:**
- [ ] Remove all author identifying information
- [ ] Abstract 100-150 words (JES: 160 OK, IJF/JEL: trim to 120)
- [ ] Keywords: 5-8 relevant terms
- [ ] All references formatted correctly (APA or numbered)
- [ ] Tables: Titled, footnoted, referenced in text
- [ ] Figures: High-resolution (300 dpi minimum), captioned
- [ ] Word count verified (range depends on journal)
- [ ] No hyperlinks or colored text in PDF
- [ ] Header/footer removed
- [ ] Page numbers consecutive
- [ ] One-inch margins all sides
- [ ] 1.5 line spacing
- [ ] No header with "Running title: ..."

**JOURNAL-SPECIFIC:**
- [ ] JES: Include JEL codes, all tables/figures embedded
- [ ] IJF: Include practitioner focus, decision trees
- [ ] JEL: Include critical assessment, only 4 main tables
